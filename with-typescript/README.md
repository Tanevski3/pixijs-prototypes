# base-pixijs-typescript-prototype

> Basic PixiJS Typescript Prototype

## Install

```sh
$ npm install
```

## Build Development

```sh
$ npm run build-dev
```

or

```sh
$ npm run watch-dev
```

## Usage

```sh
$ npm run serve-dev
```

** Open http://localhost:8080 in browser **
** Click on dist/ directory **
